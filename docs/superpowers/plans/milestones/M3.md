## MILESTONE 3 — Allowlist de egresso (rede) + prod/homolog inacessível

Objetivo: fechar o **egresso de rede** dos ambientes por padrão. Todo container roda numa **rede interna** (`--internal`, sem rota default) cujo único endpoint alcançável fora do container é um **proxy HTTP/HTTPS CONNECT** rodando no processo do produto, que só deixa passar destinos numa **allowlist por sufixo de domínio** (default `api.anthropic.com` + registries; extensível por um `sbx.allow` do projeto). Resultado: `api.anthropic.com` e os registries funcionam; **qualquer endpoint de prod/homolog é inacessível por rede**, e uma **credencial vazada no contexto do agente não consegue sair** — nem via proxy (o destino é negado), nem por conexão direta (a rede interna não tem rota). Preserva todos os invariantes de M1/M2 (roots próprios, sem socket do host, backend não vaza, portas dinâmicas). **Não** é um motor de validação declarativa — é só política de rede (invariante §7 preservado).

Este milestone entrega e **prova** o mecanismo (proxy + allowlist + rede interna + injeção via `EnvSpec.EnvVars`). O **dono do ciclo de vida do proxy** (um proxy de longa duração por sessão, iniciado junto do supervisor e cabeado no create path da CLI) é o supervisor de M4/M5: M3 forna a costura reutilizável (`netpolicy.StartProxy`/`Addr`/`Stop`/`ProxyEnv`/`LoadAllow`, rede interna no driver) e um teste de prova em processo vivo, exatamente como o supervisor a consumirá. Ver "Suposições cross-milestone" na Aceitação.

**Consome de M2 (exatamente como M2 define):**
- `EnvSpec.EnvVars map[string]string` — env vars injetadas nos containers (o proxy é injetado aqui).
- `EnvSpec.Networks []string` — redes extras a anexar (a costura de M3 pode anexar redes adicionais sem quebrar o contrato).
- `containerSpec{name,session,namespace,image,network,extraNets,envVars,publishAll}` + `createArgs(cs)` + `networkCreateArgs(net)` + `sortedKeys` (argv determinístico) — M3 estende `containerSpec` com `addHostGateway` e torna `networkCreateArgs` **interna**.
- `naming.Network(namespace)` = `<namespace>-net` (a rede própria do namespace, que passa a ser `--internal`).

**Pré-requisito de ambiente:** além do Podman rootless de M1/M2, o teste de prova (3.4) exige egresso real do **host** para puxar a imagem `curlimages/curl` e alcançar `api.anthropic.com` (o pull acontece na rede do host, não na rede interna do container). Sem podman ou sem egresso do host, o teste faz `t.Skip`. O endurecimento nftables (3.5) é **opcional** e exige `nft` + privilégio no netns; sem eles, faz `t.Skip` (o gerador de regras puro roda sempre).

**Estratégia de teste:** o proxy e a allowlist são testados **unitariamente e offline**: `matchDomain`/`allowed` são funções puras; o encaminhamento allow/deny é exercido via `httptest` (HTTP absoluto) e via **CONNECT cru** contra um `net.Listener` local (sem sair da máquina, sem container). O `LoadAllow`/`DefaultAllow`/`ProxyEnv` são puros (arquivo temporário / map). Os builders de argv do driver (`networkCreateArgs` interna, `createArgs` com `--add-host`) são unitários determinísticos. A **prova de ponta-a-ponta** (rede interna real + `curl` de dentro de um container) fica atrás de `//go:build integration`, usa `newTestPodman(t)` e `t.Skip` se `podman` ausente. Onde o comportamento do rootless netavark precisa ser confirmado rodando (alcance de `host.containers.internal` numa rede `--internal`, resolução de `host-gateway`, discriminador de deny do proxy no `curl`), o código traz o melhor esforço runnable + um marcador `> ⚠️ CONFIRMAR EM RUNTIME:` com o shape/comportamento esperado e o comando de validação.

```go
// sbx/internal/netpolicy/proxy.go
package netpolicy

import (
	"fmt"
	"io"
	"net"
	"net/http"
	"strings"
	"time"
)

// containerHostAlias is the DNS name podman injects into containers pointing at
// the host. On an --internal network it's the ONLY endpoint the container can
// reach off-box, and that endpoint is this proxy.
const containerHostAlias = "host.containers.internal"

// Proxy is an in-process HTTP/HTTPS(CONNECT) forward proxy that only permits
// egress to a suffix-matched domain allowlist. Everything else is denied
// (deny-by-default): a leaked prod/homolog hostname simply cannot be reached
// from inside a sandbox env, credential or not.
type Proxy struct {
	allow []string
	ln    net.Listener
	srv   *http.Server
	port  int
}

// StartProxy binds a dynamic host port on 0.0.0.0 (so the container can reach it
// through the internal-network gateway) and serves until Stop.
func StartProxy(allow []string) (*Proxy, error) {
	ln, err := net.Listen("tcp", "0.0.0.0:0")
	if err != nil {
		return nil, fmt.Errorf("netpolicy: listen: %w", err)
	}
	p := &Proxy{
		allow: normalizeAllow(allow),
		ln:    ln,
		port:  ln.Addr().(*net.TCPAddr).Port,
	}
	p.srv = &http.Server{Handler: p}
	go func() { _ = p.srv.Serve(ln) }()
	return p, nil
}

// Addr is the value to inject into a container as HTTP(S)_PROXY. It uses the
// podman host alias because the container sits on an --internal network whose
// only reachable off-container endpoint is the host (where this proxy listens).
func (p *Proxy) Addr() string {
	return fmt.Sprintf("http://%s:%d", containerHostAlias, p.port)
}

func (p *Proxy) Stop() error { return p.srv.Close() }

// ProxyEnv builds the four proxy env vars a container needs so every HTTP/HTTPS
// client routes egress through the proxy. Injected via EnvSpec.EnvVars (M2).
func ProxyEnv(addr string) map[string]string {
	return map[string]string{
		"HTTP_PROXY":  addr,
		"HTTPS_PROXY": addr,
		"http_proxy":  addr,
		"https_proxy": addr,
	}
}

func (p *Proxy) ServeHTTP(w http.ResponseWriter, r *http.Request) {
	if r.Method == http.MethodConnect {
		p.handleConnect(w, r)
		return
	}
	p.handleHTTP(w, r)
}

// handleConnect tunnels HTTPS after checking the allowlist. On deny it answers
// 403 BEFORE dialing, so a denied host is never contacted.
func (p *Proxy) handleConnect(w http.ResponseWriter, r *http.Request) {
	if !p.allowed(r.Host) {
		http.Error(w, "sandbox egress denied: "+hostOnly(r.Host), http.StatusForbidden)
		return
	}
	dst, err := net.DialTimeout("tcp", r.Host, 10*time.Second)
	if err != nil {
		http.Error(w, err.Error(), http.StatusBadGateway)
		return
	}
	hj, ok := w.(http.Hijacker)
	if !ok {
		http.Error(w, "proxy: hijack unsupported", http.StatusInternalServerError)
		_ = dst.Close()
		return
	}
	src, _, err := hj.Hijack()
	if err != nil {
		_ = dst.Close()
		return
	}
	_, _ = io.WriteString(src, "HTTP/1.1 200 Connection Established\r\n\r\n")
	done := make(chan struct{}, 2)
	cp := func(a, b net.Conn) { _, _ = io.Copy(a, b); done <- struct{}{} }
	go cp(dst, src)
	go cp(src, dst)
	<-done // primeira ponta a terminar fecha ambas, desbloqueando a outra cópia
	_ = src.Close()
	_ = dst.Close()
	<-done
}

// handleHTTP forwards a plain (non-CONNECT) absolute-URI request after checking
// the allowlist. On deny it answers 403 without forwarding.
func (p *Proxy) handleHTTP(w http.ResponseWriter, r *http.Request) {
	if !p.allowed(r.Host) {
		http.Error(w, "sandbox egress denied: "+hostOnly(r.Host), http.StatusForbidden)
		return
	}
	r.RequestURI = ""
	resp, err := http.DefaultTransport.RoundTrip(r)
	if err != nil {
		http.Error(w, err.Error(), http.StatusBadGateway)
		return
	}
	defer resp.Body.Close()
	for k, vs := range resp.Header {
		for _, v := range vs {
			w.Header().Add(k, v)
		}
	}
	w.WriteHeader(resp.StatusCode)
	_, _ = io.Copy(w, resp.Body)
}

func (p *Proxy) allowed(hostport string) bool {
	host := strings.ToLower(strings.TrimSuffix(hostOnly(hostport), "."))
	for _, d := range p.allow {
		if matchDomain(host, d) {
			return true
		}
	}
	return false
}

// matchDomain implements suffix-match. A bare domain matches itself and any
// subdomain (dot-bounded). A "*.foo" pattern matches subdomains only.
func matchDomain(host, pattern string) bool {
	host = strings.ToLower(strings.TrimSuffix(host, "."))
	pattern = strings.ToLower(pattern)
	if strings.HasPrefix(pattern, "*.") {
		suffix := pattern[1:] // ".ubuntu.com"
		return strings.HasSuffix(host, suffix) && len(host) > len(suffix)
	}
	return host == pattern || strings.HasSuffix(host, "."+pattern)
}

func hostOnly(hostport string) string {
	if h, _, err := net.SplitHostPort(hostport); err == nil {
		return h
	}
	return hostport
}

func normalizeAllow(in []string) []string {
	out := make([]string, 0, len(in))
	for _, d := range in {
		if d = strings.ToLower(strings.TrimSpace(d)); d != "" {
			out = append(out, d)
		}
	}
	return out
}
```

```go
// sbx/internal/netpolicy/allow.go
package netpolicy

import (
	"bufio"
	"os"
	"strings"
)

// DefaultAllow is the built-in egress allowlist: the Anthropic API plus the
// package registries an agent needs to install deps. Everything else is denied.
// Suffix-matched by matchDomain, so "api.anthropic.com" also covers subdomains.
func DefaultAllow() []string {
	return []string{
		"api.anthropic.com",
		"registry.npmjs.org",
		"pypi.org",
		"files.pythonhosted.org",
		"crates.io",
		"static.crates.io",
		"deb.debian.org",
		"*.ubuntu.com",
	}
}

// LoadAllow reads a per-project sbx.allow file (one domain per line; '#' comments
// and blank lines ignored) and merges it with DefaultAllow, deduped. A missing
// file is not an error — you just get the defaults. This is network policy only,
// never a declarative-validation engine (invariant §7).
func LoadAllow(path string) ([]string, error) {
	merged := DefaultAllow()
	f, err := os.Open(path)
	if err != nil {
		if os.IsNotExist(err) {
			return dedup(merged), nil
		}
		return nil, err
	}
	defer f.Close()

	sc := bufio.NewScanner(f)
	for sc.Scan() {
		line := strings.TrimSpace(sc.Text())
		if line == "" || strings.HasPrefix(line, "#") {
			continue
		}
		merged = append(merged, strings.ToLower(line))
	}
	if err := sc.Err(); err != nil {
		return nil, err
	}
	return dedup(merged), nil
}

func dedup(in []string) []string {
	seen := make(map[string]bool, len(in))
	out := make([]string, 0, len(in))
	for _, d := range in {
		if d == "" || seen[d] {
			continue
		}
		seen[d] = true
		out = append(out, d)
	}
	return out
}
```

```go
func curlOnNet(t *testing.T, p *Podman, network string, env []string, curlArgs ...string) (int, string) {
	t.Helper()
	args := append(p.baseArgs(), "run", "--rm",

	case err != nil:

	}
```

```go
// sbx/internal/netpolicy/hardening.go
package netpolicy

import (
	"context"
	"fmt"
	"os/exec"
	"strings"
)

// HardeningRules renders a kernel-level backstop: inside the sandbox network
// namespace, DROP all egress by default and ACCEPT only (a) loopback, (b)
// established/related return traffic, and (c) new connections to the proxy
// host:port. Defense in depth behind the --internal network + allowlist proxy.
// Pure: returns the ruleset text; applying it is ApplyHardening.
func HardeningRules(proxyHostIP string, proxyPort int) string {
	var b strings.Builder
	b.WriteString("table inet sbx_egress {\n")
	b.WriteString("  chain output {\n")
	b.WriteString("    type filter hook output priority 0; policy drop;\n")
	b.WriteString("    oif \"lo\" accept\n")
	b.WriteString("    ct state established,related accept\n")
	if proxyPort > 0 && proxyPort < 65536 && proxyHostIP != "" {
		fmt.Fprintf(&b, "    ip daddr %s tcp dport %d accept\n", proxyHostIP, proxyPort)
	}
	b.WriteString("  }\n")
	b.WriteString("}\n")
	return b.String()
}

// ApplyHardening pipes the ruleset into `nft -f -`. Requires nft on PATH and
// privilege inside the target netns; callers guard with a capability check.
func ApplyHardening(ctx context.Context, proxyHostIP string, proxyPort int) error {
	if _, err := exec.LookPath("nft"); err != nil {
		return fmt.Errorf("netpolicy: nft not found: %w", err)
	}
	cmd := exec.CommandContext(ctx, "nft", "-f", "-")
	cmd.Stdin = strings.NewReader(HardeningRules(proxyHostIP, proxyPort))
	var errb strings.Builder
	cmd.Stderr = &errb
	if err := cmd.Run(); err != nil {
		return fmt.Errorf("netpolicy: nft apply: %s: %w", strings.TrimSpace(errb.String()), err)
	}
	return nil
}
```

- Pontos marcados `⚠️ CONFIRMAR EM RUNTIME` (alcance de `host.containers.internal`/`host-gateway` na rede interna; publicação de portas de M2 sob `--internal`; discriminador de deny do `curl`; tag da imagem curl; sintaxe/aplicação do `nft`) validados na máquina-alvo e, se necessário, ajustados conforme o fallback documentado.

**Suposições cross-milestone:**
- **M3 entrega o mecanismo; M4/M5 são donos do ciclo de vida.** Os testes de prova (3.4) sobem `StartProxy` **in-process** só para validar allow/deny + rede interna. Em produção, um único proxy longo por sessão vive no `session supervise` (M4) ou no `sbx shell` (M5); o create path só lê `ProxyAddr` do registry e injeta `ProxyEnv` (M4.5).
- **Sem proxy ativo após M3:** containers em rede `--internal` ficam sem egresso (deny-by-default). Isso é intencional até `session start` / `sbx shell`.
- **M2:** `EnvSpec.EnvVars`/`Networks` e `containerSpec` são consumidos sem alteração de contrato; M3 só adiciona `addHostGateway` + `--internal`.
- **nftables (3.5):** opcional; a Aceitação de M3 **não** depende dele (Fase 2 Q1).
